Категорія Команда Опис
Управління файлами та каталогами `cd` Перейти в каталог
`ls` Перелік файлів та каталогів
Робота з текстом `echo` Вивести текст
`printf` Вивести форматований текст
`scanf` Ввести текст з клавіатури
Робота з числами `abs` Обчислити абсолютне значення числа
`pow` Обчислити степінь числа
`sqrt` Обчислити квадратний корінь числа
Робота з мережею `ping` Перевірити зв'язок із сервером
`telnet` Підключитися до сервера
`ftp` Завантажити або завантажити файл з сервера
Робота з пам'яттю `malloc` Виділити пам'ять
`free` Звільнити пам'ять
Робота з часом `time` Виміряти час виконання програми
`sleep` Затримати виконання програми
Робота з випадковими числами `rand` Вивести випадкове число
`srand` Ініціалізувати генератор випадкових чисел
Робота з файлами даних `fopen` Відкрити файл
`fclose` Закрити файл
Інші команди `sqlite3` Інтерфейс для роботи з базами даних SQLite
`dot` Програма для створення графічних зображень графів
